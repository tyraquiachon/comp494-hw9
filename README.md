## Built-In Objects and Methods Used
- `new Date()` —> creates a Date object for the current date
- `Date.getMonth()` —> gets the current month (0-based)
- `Date.getDate()` —> gets the current day
- `Date.getFullYear()` —> gets the current year
- `Number()` —> converts a value to a number
- `Number.isNaN()` —> checks if a value is NaN
- `Number.isInteger()` —> checks if a value is a whole number
- `toFixed()` —> rounds a number to a set number of decimal places
